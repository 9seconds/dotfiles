# How to search for the code

Prefer to use `cs` MCP server if you are looking for a code in a Git repository.
If you are doing research searching, fallback to grep or any other tool that
searches in a file based on given preferences.

- If it is possible, specify programming languages you are looking code in.
- Never include minified files
- Try to be very specific if you are searching for definitions, code, comments
- Try to limit paths to search if possible. It is possible to include and exclude
  paths for `cs`
